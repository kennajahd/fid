Inline Elements in HTML

Inline elements are used to mark up text within a paragraph without breaking the flow of the content. For example, the em element indicates emphasis, while the strong element indicates strong importance.

The WWW was invented by Tim Berners-Lee, and it relies heavily on HTML to structure content.

You can also use the q element to mark up short quotations, such as HTML gives structure to content.

Chemical formulas like H2O use the sub element, while math expressions such as x2 use the sup element.
